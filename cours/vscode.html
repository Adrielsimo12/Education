Introduction à VS CodePourquoi utiliser Visual Studio Code ?

Qu'est-ce que VS Code ?

Visual Studio Code (VS Code) est un éditeur de code gratuit et open-source développé par Microsoft. Il supporte des dizaines de langages (HTML, CSS, JavaScript, Python, C++, etc.) et dispose d'un écosystème d'extensions très riche.

Rapide & léger
Démarrage instantané, faible consommation RAM
Bien plus rapide que les IDE lourds
Extensions
Marketplace avec 30 000+ extensions
Python, Live Server, Prettier...
Git intégré
Gestion de version sans quitter l'éditeur
Commit, push, pull directement
Terminal intégré
Exécuter des commandes sans changer de fenêtre
npm, python, git...

Installation

Téléchargez VS Code depuis le site officiel : code.visualstudio.com

Système Installation
Windows Télécharger .exe et installer
macOS Télécharger .dmg et glisser dans Applications
Linux (Ubuntu/Debian) sudo snap install code --classic
Pourquoi VS Code ?
  • Gratuit, open-source et multiplateforme (Windows, macOS, Linux)
  • IntelliSense : auto-complétion intelligente du code
  • Débogage intégré pour JavaScript, Python, C++, etc.
  • Interface personnalisable : thèmes, icônes, raccourcis

Exercices & Travaux PratiquesProductivité maximale avec VS Code : raccourcis, extensions et debugging

VS Code est bien plus qu'un éditeur de texte. Maîtriser ses fonctionnalités avancées peut doubler votre productivité. Ces exercices vous guident vers une utilisation experte.

📝 Exercice 1 — Raccourcis clavier essentiels Facile

  1. Apprenez et pratiquez ces 20 raccourcis jusqu'à les maîtriser instinctivement : Ctrl+P (fichier rapide), Ctrl+Shift+P (palette), Ctrl+D (sélection suivante), Alt+Click (multi-curseur), Ctrl+/ (commentaire), Alt+↑/↓ (déplacer ligne), Ctrl+B (sidebar), F12 (aller à définition), Ctrl+Shift+K (supprimer ligne), Ctrl+` (terminal).
  2. Créez votre propre keybindings.json avec 3 raccourcis personnalisés pour vos actions les plus fréquentes.

📝 Exercice 2 — Débogage intégré Moyen

  1. Configurez le débogueur VS Code pour Python : créez un fichier .vscode/launch.json avec une configuration qui démarre votre script Flask en mode debug.
  2. Mettez des points d'arrêt, utilisez la console de débogage, inspectez les variables locales et globales, parcourez le code pas à pas (F10, F11, Shift+F11).
  3. Configurez le débogueur JavaScript pour Node.js et déboguez une fonction asynchrone complexe.
  4. Utilisez les logpoints (points d'arrêt sans pause, qui logguent seulement) pour déboguer en production sans arrêter l'exécution.

🔬 TP — Configuration d'un workspace professionnel Avancé — 2h

Objectif : Configurer VS Code pour un projet web full-stack avec qualité de code automatisée.

  1. Extensions indispensables : Installer et configurer : ESLint, Prettier, Python, GitLens, Thunder Client, Live Share, Error Lens, Auto Rename Tag.
  2. Settings.json : Configurer le formattage automatique à la sauvegarde, l'indentation (2 espaces), la règle de 80 caractères, le thème et les icônes.
  3. Workspace settings : Créer .vscode/settings.json avec les configs spécifiques au projet (Python path, lint rules).
  4. Snippets : Créer 5 snippets personnalisés pour vos patterns de code les plus utilisés (ex: composant Flask, structure HTML5, classe Python).
  5. Tasks : Configurer .vscode/tasks.json avec des tâches pour : lancer le serveur Flask, lancer les tests, builder le CSS.
