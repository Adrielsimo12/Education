À propos de ce cours

Ce cours est conçu pour les étudiants de la filière CIEL (Cybersécurité, Informatique et réseaux, Électronique). Il couvre trois domaines complémentaires : le développement web Full Stack, l'électronique et les microcontrôleurs, et les réseaux informatiques & la cybersécurité.

Chaque chapitre est structuré avec des explications détaillées, des exemples de code fonctionnels, et des exercices pratiques avec corrigés. L'objectif est de vous rendre autonome dans la création d'applications web complètes, dans la conception de circuits électroniques et dans la compréhension des infrastructures réseau.

Conseil : Suivez les cours dans l'ordre proposé. Chaque chapitre s'appuie sur les connaissances acquises dans les précédents. Dans chaque cours, cliquez sur "Voir la réponse" pour afficher les corrections des exercices après avoir essayé par vous-même !

Développement Web Full StackDu HTML au déploiement d'une application complète

Qu'est-ce que le développement Full Stack ?

Un développeur Full Stack maîtrise à la fois le frontend (ce que voit l'utilisateur) et le backend (la logique serveur et les données). Ce cours vous apprendra les deux aspects, de la mise en forme d'une page HTML jusqu'à la création d'une API sécurisée avec base de données.

Frontend
L'interface utilisateur visible dans le navigateur
HTML • CSS • JavaScript
Backend
Le serveur qui traite les données et la logique métier
Python • Flask • API REST
Base de données
Stockage persistant et structuré des informations
MySQL • SQL • PyMySQL

PARTIE 2
Électronique & MicrocontrôleursDes composants de base aux projets IoT connectés

Pourquoi l'électronique dans la filière CIEL ?

La filière CIEL forme des techniciens capables d'intervenir sur des systèmes connectés qui combinent informatique et électronique. Comprendre les composants et savoir programmer des microcontrôleurs est essentiel pour créer des objets connectés (IoT) et diagnostiquer des pannes matérielles.

IoT (Internet of Things) : objets physiques connectés à Internet, capables de collecter et transmettre des données. Exemples : thermostats intelligents, capteurs industriels, systèmes de domotique, véhicules autonomes.

PARTIE 3
Réseaux & CybersécuritéComprendre, configurer et sécuriser les infrastructures réseau

Pourquoi les réseaux dans la filière CIEL ?

La lettre R de CIEL signifie Réseaux. Tout système informatique ou électronique connecté repose sur une infrastructure réseau. Comprendre les protocoles, les équipements et les menaces est fondamental pour concevoir des systèmes fiables, sécurisés et performants.

Cette partie vous permettra de comprendre comment les données transitent d'un appareil à l'autre à travers Internet, comment les administrer et comment les protéger contre les attaques.

Éthique : Les connaissances en cybersécurité doivent être utilisées à des fins défensives et légales uniquement. Les tests d'intrusion ne sont autorisés que sur des systèmes dont vous êtes propriétaire ou sur lesquels vous avez une autorisation écrite explicite.

Lien avec les autres parties

Domaine Connexion avec les réseaux
Développement Web Les applications Flask/MySQL communiquent via HTTP/HTTPS — les notions de réseau permettent de comprendre les requêtes, les ports, le déploiement sur serveur.
Électronique IoT Un ESP32 connecté en WiFi utilise TCP/IP, UDP, MQTT. Les capteurs envoient des données via des protocoles réseau sécurisés.
Base de données MySQL écoute sur le port 3306. Les connexions distantes doivent être sécurisées (SSL, pare-feu, comptes utilisateurs MySQL limités).
Ce que vous saurez faire après la Partie 3
  • Expliquer le modèle OSI et les protocoles TCP/IP
  • Configurer l'adressage IP, les masques de sous-réseau et les VLAN
  • Utiliser Wireshark pour capturer et analyser du trafic réseau
  • Mettre en place un pare-feu et des règles de filtrage
  • Comprendre la cryptographie (symétrique, asymétrique, hachage)
  • Identifier et corriger les vulnérabilités courantes (OWASP Top 10)
  • Simuler une infrastructure réseau avec Cisco Packet Tracer
