Qu'est-ce qu'un Raspberry Pi ?

Le Raspberry Pi est un nano-ordinateur de la taille d'une carte de crédit, développé par la Fondation Raspberry Pi. Il possède un processeur ARM, de la RAM, des ports USB, HDMI, Ethernet et surtout : des broches GPIO (General Purpose Input/Output) pour contrôler des composants électroniques.

Modèles populaires : Raspberry Pi 4 (jusqu'à 8 Go RAM), Raspberry Pi 3B+, Raspberry Pi Zero (ultra compact), Raspberry Pi Pico (microcontrôleur).

Comparaison Raspberry Pi vs Arduino

Critère Raspberry Pi Arduino
Type Ordinateur complet (OS Linux) Microcontrôleur (firmware)
Puissance Haute (multi-tâches, navigateur web) Faible (une tâche à la fois)
Prix 35€ - 75€ 5€ - 25€
Langage Python, C++, JavaScript, etc. C/C++ (Arduino IDE)
GPIO ✓ Oui (40 broches) ✓ Oui (digital/analog pins)
Usage Serveur web, IA, multimedia, IoT Capteurs temps réel, robotique
Points clés
  • Le Raspberry Pi est un mini-ordinateur avec un système d'exploitation Linux
  • Idéal pour des projets IoT, serveurs web, domotique et apprentissage machine
  • Possède 40 broches GPIO pour contrôler LED, moteurs, capteurs
  • Se programme principalement en Python (RPi.GPIO, gpiozero)

RPI.03
Les broches GPIOContrôle d'entrées et sorties numériques

Schéma des GPIO (40 broches)

Le Raspberry Pi possède 40 broches GPIO numérotées. Certaines sont dédiées à l'alimentation (3.3V, 5V, GND), d'autres au contrôle (GPIO numérotés de 2 à 27).

Numérotation : Il existe deux systèmes : numérotation physique (1-40) et numérotation BCM (GPIO 2, GPIO 3, etc.). En Python, on utilise généralement BCM.
Type de broche Fonction Exemple
3.3V Alimentation 3.3V Alimenter capteurs logiques
5V Alimentation 5V Moteurs, LED puissantes
GND (Ground) Masse / 0V Référence commune
GPIO Entrée/Sortie numérique Contrôle LED, lecture bouton
I2C / SPI / UART Protocoles de communication Écran OLED, capteurs avancés

Allumer une LED avec Python

Python 
# Installation de la librairie GPIO
# sudo apt install python3-rpi.gpio

import RPi.GPIO as GPIO
import time

# Configuration
GPIO.setmode(GPIO.BCM)  # Numérotation BCM
GPIO.setwarnings(False)
LED_PIN = 17  # GPIO 17 (broche physique 11)

# Définir GPIO 17 en sortie
GPIO.setup(LED_PIN, GPIO.OUT)

# Allumer la LED
GPIO.output(LED_PIN, GPIO.HIGH)
time.sleep(2)

# Éteindre la LED
GPIO.output(LED_PIN, GPIO.LOW)

# Nettoyer les GPIO
GPIO.cleanup()

Faire clignoter une LED (blink)

Python 
import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
LED_PIN = 17
GPIO.setup(LED_PIN, GPIO.OUT)

try:
    while True:
        GPIO.output(LED_PIN, GPIO.HIGH)  # Allumer
        time.sleep(0.5)
        GPIO.output(LED_PIN, GPIO.LOW)   # Éteindre
        time.sleep(0.5)
except KeyboardInterrupt:
    GPIO.cleanup()  # Arrêter proprement avec Ctrl+C
RPI.04
Lecture d'entréesBoutons, interrupteurs et résistances pull-up/down

Lire l'état d'un bouton

Pour lire un signal d'entrée (bouton, capteur numérique), on configure la broche en GPIO.IN et on utilise GPIO.input().

Python 
import RPi.GPIO as GPIO

GPIO.setmode(GPIO.BCM)
BUTTON_PIN = 27  # GPIO 27

# Activer résistance pull-up interne (bouton = LOW quand appuyé)
GPIO.setup(BUTTON_PIN, GPIO.IN, pull_up_down=GPIO.PUD_UP)

while True:
    button_state = GPIO.input(BUTTON_PIN)
    if button_state == GPIO.LOW:
        print("Bouton appuyé !")
    else:
        print("Bouton relâché")
    time.sleep(0.1)

Résistances Pull-Up et Pull-Down

Type Comportement Usage
PUD_UP Broche à 3.3V par défaut (HIGH) Bouton connecté à GND → LOW quand appuyé
PUD_DOWN Broche à 0V par défaut (LOW) Bouton connecté à 3.3V → HIGH quand appuyé

Projet 1 : Feu tricolore automatique

Python 
import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)

# Broches des LED
LED_ROUGE = 17
LED_ORANGE = 27
LED_VERTE = 22

# Configuration en sortie
GPIO.setup(LED_ROUGE, GPIO.OUT)
GPIO.setup(LED_ORANGE, GPIO.OUT)
GPIO.setup(LED_VERTE, GPIO.OUT)

def feu_tricolore():
    while True:
        # Feu ROUGE
        GPIO.output(LED_ROUGE, GPIO.HIGH)
        time.sleep(5)
        
        # Feu ORANGE
        GPIO.output(LED_ROUGE, GPIO.LOW)
        GPIO.output(LED_ORANGE, GPIO.HIGH)
        time.sleep(2)
        
        # Feu VERT
        GPIO.output(LED_ORANGE, GPIO.LOW)
        GPIO.output(LED_VERTE, GPIO.HIGH)
        time.sleep(5)
        
        # Retour ORANGE
        GPIO.output(LED_VERTE, GPIO.LOW)
        GPIO.output(LED_ORANGE, GPIO.HIGH)
        time.sleep(2)
        GPIO.output(LED_ORANGE, GPIO.LOW)

try:
    feu_tricolore()
except KeyboardInterrupt:
    GPIO.cleanup()

Projet 2 : Station météo avec capteur DHT11

Python 
# Installation : sudo pip3 install Adafruit_DHT

import Adafruit_DHT
import time

# Type de capteur et broche GPIO
SENSOR = Adafruit_DHT.DHT11
PIN = 4  # GPIO 4

while True:
    humidite, temperature = Adafruit_DHT.read_retry(SENSOR, PIN)
    
    if humidite is not None and temperature is not None:
        print(f"Température : {temperature:.1f}°C")
        print(f"Humidité : {humidite:.1f}%")
    else:
        print("Erreur de lecture du capteur")
    
    time.sleep(2)

Exercices & Travaux PratiquesAdministration Linux, GPIO, serveurs et projets Raspberry Pi

Le Raspberry Pi combine puissance d'un ordinateur Linux et flexibilité d'un microcontrôleur. Ces exercices couvrent l'administration système, la programmation GPIO et les projets serveurs.

📝 Exercice 1 — Administration Linux de base Facile

  1. Installez Raspberry Pi OS Lite. Configurez le WiFi, activez SSH, et connectez-vous via ssh pi@adresse_ip depuis votre PC.
  2. Maîtrisez les commandes essentielles : ls -la, cd, mkdir, cp, mv, rm -rf, chmod 755, chown, top, df -h, free -m.
  3. Créez un script bash maintenance.sh qui : met à jour le système (apt update && apt upgrade -y), affiche l'utilisation CPU/RAM/disque, et envoie un rapport par email (avec msmtp).
  4. Programmez ce script avec crontab -e pour s'exécuter tous les dimanches à 3h du matin.

📝 Exercice 2 — GPIO avec Python Moyen

  1. Utilisez la bibliothèque RPi.GPIO pour contrôler une LED (GPIO 18) et lire un bouton (GPIO 24, mode pull-up interne).
  2. Implémentez une LED RGB avec PWM sur 3 broches GPIO. Créez une fonction set_color(r, g, b) et animez des transitions de couleur douces.
  3. Créez un serveur Flask sur le Raspberry Pi qui expose une API pour contrôler les LEDs depuis n'importe quel appareil du réseau local.

🔬 TP — Serveur multimédia personnel (NAS + Streaming) Avancé — 4h

Objectif : Transformer un Raspberry Pi en serveur multimédia accessible depuis tout le réseau local.

  1. NAS basique : Monter un disque USB (auto-mount au démarrage via /etc/fstab), partager via Samba pour accès Windows/Mac/Linux.
  2. Streaming vidéo : Installer Jellyfin ou Plex Media Server. Configurer les bibliothèques de films/séries/musique.
  3. IP statique : Assigner une IP fixe au Raspberry Pi via la box internet ou en modifiant dhcpcd.conf.
  4. Surveillance : Installer Netdata pour monitorer en temps réel CPU, RAM, température, réseau.
  5. Sauvegarde : Script de sauvegarde automatique des fichiers importants vers le cloud (rclone vers Google Drive).
