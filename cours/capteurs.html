CAPT.01
IntroductionCapteurs vs Actionneurs — Systèmes IoT

Qu'est-ce qu'un capteur ?

Un capteur est un composant électronique qui mesure une grandeur physique (température, lumière, pression, distance, etc.) et la convertit en signal électrique exploitable par un microcontrôleur (Arduino, ESP32, Raspberry Pi).

Exemple : Un capteur de température DHT11 mesure la température ambiante et envoie une valeur numérique (ex: 23°C) à l'Arduino via une broche GPIO.

Qu'est-ce qu'un actionneur ?

Un actionneur est un composant qui effectue une action physique (mouvement, son, lumière) en réponse à un signal électrique envoyé par le microcontrôleur.

CAPTEURS
Mesure → Signal électrique
DHT11, HC-SR04, LDR, PIR
ACTIONNEURS
Signal électrique → Action
Moteurs, servos, relais, buzzers
Points clés
  • Les capteurs collectent des données (INPUT)
  • Les actionneurs effectuent des actions (OUTPUT)
  • Un système IoT combine les deux pour automatiser des tâches
  • Exemples : thermostat intelligent, éclairage automatique, robot éviteur d'obstacles
CAPT.02
Capteur de températureDHT11, DHT22, DS18B20, LM35

DHT11 / DHT22 (Température + Humidité)

Capteur Plage température Précision Prix
DHT11 0-50°C ±2°C ~2€
DHT22 -40 à 80°C ±0.5°C ~5€
DS18B20 -55 à 125°C ±0.5°C ~3€

Code Arduino : DHT11

Arduino C++ 
// Installer : DHT sensor library by Adafruit
#include <DHT.h>

#define DHTPIN 2      // Broche DATA du DHT11
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float temp = dht.readTemperature();
  float humidity = dht.readHumidity();
  
  if (isnan(temp) || isnan(humidity)) {
    Serial.println("Erreur capteur");
    return;
  }
  
  Serial.print("Temp: ");
  Serial.print(temp);
  Serial.println(" °C");
  
  Serial.print("Humidité: ");
  Serial.print(humidity);
  Serial.println(" %");
  
  delay(2000);
}
CAPT.03
Capteur de distanceHC-SR04 Ultrason

HC-SR04 : Fonctionnement

Le capteur HC-SR04 utilise des ultrasons pour mesurer la distance. Il émet une impulsion sonore et calcule le temps de retour de l'écho.

Distance (cm) = (Temps × 340) / 2 / 10000
340 m/s = vitesse du son • Division par 2 (aller-retour)

Code Arduino : HC-SR04

Arduino C++ 
const int TRIG_PIN = 9;
const int ECHO_PIN = 10;

void setup() {
  Serial.begin(9600);
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
}

void loop() {
  // Envoyer une impulsion de 10µs
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  
  // Mesurer le temps de retour
  long duration = pulseIn(ECHO_PIN, HIGH);
  
  // Calculer la distance en cm
  float distance = (duration * 0.034) / 2;
  
  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");
  
  delay(500);
}
Application : Robot éviteur d'obstacles, parking automatique, mesure de niveau d'eau.
CAPT.04
Capteur de lumièreLDR (Light Dependent Resistor)

LDR : Photorésistance

Une LDR (photorésistance) est un composant dont la résistance varie selon l'intensité lumineuse :

  • Forte luminosité → Résistance faible (~1 kΩ)
  • Obscurité → Résistance élevée (~1 MΩ)

Code Arduino : LDR

Arduino C++ 
const int LDR_PIN = A0;
const int LED_PIN = 13;

void setup() {
  Serial.begin(9600);
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  int lightValue = analogRead(LDR_PIN);  // 0-1023
  
  Serial.print("Lumière: ");
  Serial.println(lightValue);
  
  // Allumer la LED si faible luminosité
  if (lightValue < 300) {
    digitalWrite(LED_PIN, HIGH);
  } else {
    digitalWrite(LED_PIN, LOW);
  }
  
  delay(200);
}
Application : Éclairage automatique, suivi solaire, détection jour/nuit.
ACT.01
MoteursMoteur DC, moteur pas-à-pas, servomoteur

Types de moteurs

Type Usage Contrôle
Moteur DC Rotation continue (roues, ventilateurs) PWM + Pont H (L298N)
Servomoteur Rotation précise (-90° à +90°) Signal PWM
Moteur pas-à-pas Rotation ultra-précise (imprimante 3D) Driver A4988

Code Arduino : Servomoteur

Arduino C++ 
#include <Servo.h>

Servo monServo;

void setup() {
  monServo.attach(9);  // Broche 9
}

void loop() {
  monServo.write(0);     // Position 0°
  delay(1000);
  
  monServo.write(90);    // Position 90°
  delay(1000);
  
  monServo.write(180);   // Position 180°
  delay(1000);
}

Code Arduino : Moteur DC avec L298N

Arduino C++ 
// Connexions L298N
const int ENA = 9;   // PWM (vitesse)
const int IN1 = 7;   // Direction
const int IN2 = 6;

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
}

void loop() {
  // Avancer à 50% de vitesse
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 128);  // 50% = 128/255
  delay(2000);
  
  // Arrêter
  analogWrite(ENA, 0);
  delay(1000);
  
  // Reculer
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  analogWrite(ENA, 128);
  delay(2000);
}
ACT.02
RelaisContrôle d'appareils haute tension (230V)

Qu'est-ce qu'un relais ?

Un relais est un interrupteur électromécanique contrôlé par un signal électrique faible (5V Arduino) qui permet de contrôler des charges haute tension (230V lampe, ventilateur, etc.).

Sécurité : Manipuler le 230V est dangereux. Toujours débrancher avant câblage. Si vous n'êtes pas sûr, demandez l'aide d'un électricien.

Code Arduino : Relais

Arduino C++ 
const int RELAY_PIN = 8;

void setup() {
  pinMode(RELAY_PIN, OUTPUT);
}

void loop() {
  // Allumer l'appareil (fermer le relais)
  digitalWrite(RELAY_PIN, HIGH);
  delay(5000);
  
  // Éteindre l'appareil (ouvrir le relais)
  digitalWrite(RELAY_PIN, LOW);
  delay(5000);
}
Application : Domotique (allumer/éteindre lampe, cafetière, chauffage via smartphone).
PROJ
Projet completSystème d'arrosage automatique

Description du projet

Un système d'arrosage automatique qui mesure l'humidité du sol et active une pompe si le sol est trop sec. Utilise un capteur d'humidité du sol et un relais.

Code Arduino

Arduino C++ 
const int SOIL_PIN = A0;      // Capteur humidité sol
const int PUMP_RELAY = 8;   // Relais pompe
const int THRESHOLD = 500;   // Seuil sec/humide

void setup() {
  Serial.begin(9600);
  pinMode(PUMP_RELAY, OUTPUT);
  digitalWrite(PUMP_RELAY, LOW);  // Pompe éteinte
}

void loop() {
  int soilMoisture = analogRead(SOIL_PIN);
  
  Serial.print("Humidité sol: ");
  Serial.println(soilMoisture);
  
  if (soilMoisture < THRESHOLD) {
    Serial.println("Sol sec → Pompe ON");
    digitalWrite(PUMP_RELAY, HIGH);
    delay(5000);  // Arroser 5 secondes
    digitalWrite(PUMP_RELAY, LOW);
  } else {
    Serial.println("Sol humide → Pompe OFF");
  }
  
  delay(10000);  // Vérifier toutes les 10 secondes
}
Pour aller plus loin
  • Ajouter un capteur DHT11 pour surveiller température et humidité ambiante
  • Utiliser un ESP32 pour envoyer les données sur un serveur web
  • Créer une interface web pour contrôler manuellement la pompe
  • Ajouter un écran LCD pour afficher les informations en temps réel

Exercices & Travaux PratiquesInterfaçage et calibration de capteurs avec microcontrôleurs

Ces exercices pratiques vous permettent de maîtriser l'interfaçage de capteurs variés (température, distance, luminosité, mouvement) avec des microcontrôleurs Arduino et ESP32.

📝 Exercice 1 — Capteur ultrasonique HC-SR04 Facile

  1. Branchez le HC-SR04 sur Arduino (Trig→broche 9, Echo→broche 10). Codez la mesure de distance et affichez-la en cm sur le moniteur série.
  2. Calibrez le capteur avec un objet à distance connue. Calculez l'erreur absolue et relative.
  3. Créez un radar de recul : LED verte si distance > 50 cm, orange entre 20-50 cm, rouge < 20 cm, buzzer si < 10 cm.
  4. Expliquez le principe physique : comment calculer la distance à partir du temps de vol d'une onde sonore ?

📝 Exercice 2 — Moteur DC et servo Moyen

  1. Contrôlez la vitesse d'un moteur DC via un transistor NPN (ou L298N) depuis Arduino avec PWM. Mappez la valeur d'un potentiomètre (0-1023) vers la vitesse (0-255).
  2. Pilotez un servo-moteur SG90 avec la bibliothèque Servo.h. Balayez de 0° à 180° et retour. Synchronisez deux servos pour un bras robotique simple (2 degrés de liberté).
  3. Combinez HC-SR04 + servo : créez un radar tournant qui scanne 180° et affiche les distances détectées.

🔬 TP — Serre connectée (Smart Garden) Avancé — 3h

Objectif : Construire un système de monitoring et d'arrosage automatique pour une plante.

  1. Capteurs : DHT22 (T°/humidité air), capteur d'humidité sol capacitif, LDR (luminosité).
  2. Actionneurs : Pompe à eau mini 5V pilotée via relais, ventilateur 5V via transistor, LED grow light via MOSFET.
  3. Logique : Arrosage automatique si humidité sol < 30%. Ventilation si T° > 28°C. Éclairage si luminosité < seuil ET heure entre 8h-20h (RTC DS3231).
  4. Affichage : LCD I2C affichant en temps réel T°, humidité, état de la pompe.
  5. Logs : Toutes les 10 minutes, écrire les données sur une carte SD (module SD). Format CSV pour exploitation tableur.
  6. Rendu : Maquette fonctionnelle + code commenté + rapport avec graphiques des données collectées sur 24h.
